# Brandon Pavao Portfolio Website

A personal portfolio website styled as a simple platformer game, where visitors move horizontally through project buildings.

This project was designed to make my portfolio feel more interactive than a standard resume page. Instead of scrolling through sections, visitors navigate through a game-like world where each building represents a project. Each project space includes documentation, development context, learning outcomes, and links to live hosted versions when available.

The goal was to present my work in a format that reflects my interest in UX, software development, and playful digital systems.

## Overview

The portfolio is built around a horizontal navigation experience inspired by side-scrolling platformer games.

Each building acts as a project entry point. Visitors can explore my work through visual spaces rather than static cards. This creates a portfolio that feels personal, organized, and memorable while still keeping project information accessible.

## Features

* Platformer-inspired portfolio layout
* Horizontal navigation system
* Project buildings used as interactive sections
* Dedicated project documentation pages
* Links to hosted projects
* Experience writeups for each project
* Visual project previews
* Personal branding through game-style presentation
* Mobile and desktop layout planning
* Static website structure for simple hosting
* Portfolio-first content organization
* Clear separation between projects, experience, and external links

## What I Learned

* How to design a portfolio around interaction instead of static scrolling
* How to turn project navigation into part of the user experience
* How to organize documentation across multiple hosted projects
* How to balance playful design with readable portfolio content
* How to create a site that reflects both UX and development skills
* How to make personal branding feel built into the interface
* How to present technical work in a format that feels more memorable
